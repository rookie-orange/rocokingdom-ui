# tokens

Four direct color values for `rocokingdom-ui`.

## Usage

```ts
import { gold, paper } from '@rocokingdom-ui/tokens'

const pageBackground = paper
const buttonBackground = gold
```

## Structure

- `paper`
- `stone`
- `ink`
- `gold`
- `colors`

## Development

- Install dependencies:

```bash
vp install
```

- Run the unit tests:

```bash
vp test
```

- Build the library:

```bash
vp run build
```
