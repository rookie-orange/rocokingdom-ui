# tokens

Material color values for `rocokingdom-ui`.

## Usage

```ts
import { paper, primary } from '@rocokingdom-ui/tokens'

const pageBackground = paper
const buttonBackground = primary
```

## Structure

- `paper`
- `stone`
- `primary`
- `onPaper`
- `onStone`
- `onPrimary`
- `colors`
- `cssVariables`

## Development

- Install dependencies:

```bash
vp install
```

- Run the unit tests:

```bash
vp test
```

- Build the library:

```bash
vp run build
```
