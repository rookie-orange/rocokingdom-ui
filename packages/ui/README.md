# ui

Rocokingdom UI components.

## Button

```ts
import 'rocokingdom-ui/style.css'
import { Button } from 'rocokingdom-ui'

export function App() {
  return <Button>Start</Button>
}
```

The button uses one class: `rk-button`.

## Development

- Install dependencies:

```bash
vp install
```

- Run the unit tests:

```bash
vp test
```

- Build the library:

```bash
vp run build
```
